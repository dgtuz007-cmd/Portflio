## 📖 Loyiha haqida

> `// portfolio.init()` 🚀

Bu — frontend dunyosiga endi qadam qo'ygan **web-dasturchi Ofoq Boboxonovning** shaxsiy portfolio sayti. Sayt terminal/kod-muharrir uslubida ishlangan: navigatsiya menyusi `$ whoami`, `$ ls projects/`, `$ stack --list` kabi terminal buyruqlari shaklida taqdim etilgan, bu esa saytga noyob va "dasturchilarcha" hissiyot beradi. 🧑‍💻

Saytning o'zi ham bir loyiha sifatida qaraladi — uni qurish jarayonida muallif sof JavaScript yordamida animatsiya, **dark/light tema almashtirish** 🌗 va to'liq **responsiv dizayn** 📱💻 ustida mashq qilgan.

## 🧩 Sayt bo'limlari

| 🔹 Bo'lim | 🖱️ Terminal buyrug'i | 📌 Tavsif |
|---|---|---|
| 🏠 Hero | `~` | Bosh sahifa, qisqa tanishtiruv va CTA tugmalar |
| 👤 Men haqimda | `$ whoami` | Dasturchi haqida ma'lumot, `about.js` kod bloki va o'rganish jurnali (`journey.log`) |
| 🗂️ Loyihalar | `$ ls projects/` | Amalga oshirilgan mini-loyihalar ro'yxati |
| 🛠️ Ko'nikmalar | `$ stack --list` | Texnologiyalar bo'yicha progress-barlar |
| ✉️ Bog'lanish | `$ contact --send` | Aloqa formasi va ijtimoiy tarmoq havolalari |

- 🌓 **Tema almashtirish** — yorug' va qorong'i rejim orasida bir tugma bosish bilan o'tish

- 🧮 **Real mini-loyihalar vitrinasi** — kalkulyator, vazifalar ro'yxati, ob-havo ilovasi va blog maketi
